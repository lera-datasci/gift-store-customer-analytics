Gift Store Customer Analytics

End-to-end customer analytics project based on transactional data from an online gift store.

The analysis includes data cleaning, exploratory data analysis (EDA), customer behavior analysis, and RFM segmentation.

Daily and weekly activity patterns were identified, while monthly seasonality was not confirmed.

Customer engagement (stickiness) remained stable at ~5% across Q2–Q3 2019.

RFM analysis highlighted high-value customer segments (Champions and Loyal customers).

Several statistical hypotheses were tested (Z-test, Mann–Whitney U test), with no significant differences found.

Tools: Python, pandas, numpy, matplotlib, seaborn, pyplot, scipy, statsmodels, Jupyter Notebook
